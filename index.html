About Project.

This our project and homework repository for ECG 604 / CPE 404. We are tasked to design a simple toy microprocessor which includes:

  • An Assembler in C
  • Implementation through an FPGA

with architecture features including:

  • Two Level Cache
  • Two Issue SS
  • Floating Point Unit
  • Branch Prediction and more

Our processor is based on a subset of the MIPS ISA, with support of the MIPS CoProc 1 subset support for floating point
